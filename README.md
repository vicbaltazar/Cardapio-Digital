# VicFelas — Cardápio Digital

> Cardápio web estático, artesanal e responsivo **VicFelas**. desenvolvido com HTML, CSS puro e JavaScript vanilla, sem dependências externas.

---

## ✨ Demonstração

Abra o arquivo `index.html` diretamente no navegador, não requer servidor ou build.

---

## 🚀 Como usar

### Visualizar localmente

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/vicfelas.git
cd vicfelas

# Abra no navegador (macOS)
open index.html

# Abra no navegador (Linux)
xdg-open index.html

# Abra no navegador (Windows)
start index.html
```

---

## 🧰 Stack

- **HTML5** semântico
- **CSS3** — Custom Properties, Grid, `color-mix()`, `clip-path`, `mix-blend-mode`
- **JavaScript vanilla** — `IntersectionObserver`, `mousemove` parallax
- **Google Fonts** — Bebas Neue + DM Sans

Sem frameworks. Sem bundlers. Sem dependências.
